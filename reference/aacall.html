galpy.actionAngle.actionAngle.__call__¶

actionAngle.__call__(*args, **kwargs)¶
NAME:
__call__
PURPOSE:
evaluate the actions (jr,lz,jz)

INPUT:

Either:

  1. R,vR,vT,z,vz:
    1. floats: phase-space value for single object
    2. numpy.ndarray: [N] phase-space values for N objects
    3. numpy.ndarray: [N,M] phase-space values for N objects at M times
  2. Orbit instance or list thereof; can be integrated already
OUTPUT:
(jr,lz,jz)
HISTORY:
2014-01-03 - Written for top level - Bovy (IAS)
